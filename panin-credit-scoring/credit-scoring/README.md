# How to Run?

Step 1:
Navigate to credit-scoring directory
> cd /path/to/credit-scoring

Step 2:
Copy .env example
> cp .env.example .env

Step 3:
Run docker compose up
> docker compose up --build


1. Mlflow: <http://localhost:5440>
2. API/SWAGGER: <http://localhost:8092/docs>
